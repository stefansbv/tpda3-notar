Installation
------------

Set DBI_USER and DBI_PASS environment vars before test.

% perl Makefile.PL
% make
% make test
% make install


Requirements
------------

- Perl v5.8.9 or newer.
- RDBMS:  PostgreSQL version 8.2 or greater
